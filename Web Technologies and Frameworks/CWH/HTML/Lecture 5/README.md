# HTML Media and Lists Practice

This lecture demonstrated the use of images, tables, and lists in HTML as part of my Web Development learning journey.

## Features

- Displaying images using the `<img>` tag
- Creating structured tables with headings and rows
- Using unordered and ordered lists
- Basic webpage structure with semantic HTML elements

## Concepts Practiced

- Image embedding
- Table creation in HTML
- Ordered and unordered lists
- HTML document structure
- Meta tags and page setup

## Technologies Used

- HTML

## Purpose

The purpose of this project is to practice fundamental HTML concepts through hands-on implementation and structured webpage creation.
