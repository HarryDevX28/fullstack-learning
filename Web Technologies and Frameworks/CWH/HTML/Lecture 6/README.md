# SEO and Core Web Vitals Practice

This Lecture focused on the fundamentals of SEO (Search Engine Optimization) and Core Web Vitals in Web Development.

The webpage was created to practice SEO-friendly HTML structure and the use of essential meta tags for better search engine visibility and user experience.

## Concepts Practiced

- SEO fundamentals
- Meta description tags
- Responsive viewport configuration
- HTML page structure
- Semantic content organization
- Introduction to Core Web Vitals

## Technologies Used

- HTML

## Purpose

The purpose of this project is to understand how proper HTML structure and metadata contribute to website accessibility, search engine optimization, and overall web performance.
